## ✨ Features

- **Blazing Fast Performance:** Powered by Vite HMR and optimized React components.
- **Dark/Light Mode:** Seamless, context-aware theme switching.
- **Responsive Design:** Fluid layouts ensuring a perfect experience on mobile, tablet, and desktop securely built with CSS Grid and Flexbox.
- **Smooth Animations:** Integrated intersection observer for scroll-reveal effects and glassmorphism UI elements.
- **Component-Driven:** Modular architecture making it simple to easily update projects, experience, and details.

## 💻 Running Locally

To get a local copy up and running, follow these simple steps:

### Prerequisites

You need to have Node.js and npm installed on your machine.
- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iakhilrt/akhilrt-portfolio.git
   ```
2. Navigate to the project directory
   ```sh
   cd akhilrt-portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
   > The application will typically start on `http://localhost:5173/`
